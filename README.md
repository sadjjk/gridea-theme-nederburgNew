## 安装
1. 如果Gridea正在运行，请先关闭.
2. 进入 **<你的Gridea主题文件夹>** 然后执行 ``` git clone https://github.com/sadjjk/gridea-theme-nederburgNew.git ```
3. 重新启动Gridea，在「主题」菜单中选择 「nederburgNew」，点击保存！
